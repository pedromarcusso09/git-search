# GitSearch

GitSearch é uma aplicação web que permite pesquisar perfis de usuários do GitHub e visualizar informações detalhadas de cada perfil.

## Funcionalidades

- Busca de usuários do GitHub pelo nome de usuário
- Exibição de informações do perfil (nome, bio, avatar, etc.)
- Tratamento de erros para usuários não encontrados
- Interface responsiva e amigável


## Como usar

1. Clone o repositório:
   ```bash
   git clone <https://github.com/pedromarcusso09/git-search>
   ```
2. Abra o arquivo `index.html` em seu navegador.
3. Pesquise por um usuário do GitHub para visualizar o perfil.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
